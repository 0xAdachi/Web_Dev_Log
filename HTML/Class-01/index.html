Markup

HTML markup consists of several key components, including those called tags (and their attributes).

Line Breaks

The tag <br> is used to break lines in a html document.

Anchor Tag

The anchor tag is used to link other html files.

Naming conventions

The most common filename extension for files containing HTML is .html
